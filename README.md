# Calendar TODO Chrome Extension

カレンダー形式のTODO管理Chrome拡張機能

## 機能

### 主要機能
- 📅 **カレンダービュー**: 月表示・週表示・日表示の3つのビューモード
- ✅ **タスク管理**: タスクの作成・編集・削除・完了管理
- 🎯 **優先度設定**: 高・中・低の3段階の優先度
- 🏷️ **カテゴリ管理**: カスタマイズ可能なカテゴリでタスクを整理
- 🔔 **リマインダー**: タスクのリマインダー通知
- 💾 **データ同期**: Chromeアカウント間でのデータ同期
- 📤 **エクスポート/インポート**: データのバックアップと復元

## インストール方法

### 開発版のインストール

1. リポジトリをクローンまたはダウンロード
```bash
git clone [repository-url]
cd ToDoChromeExtension
```

2. 依存関係をインストール
```bash
npm install
```

3. 拡張機能をビルド
```bash
npm run build
```

4. Chromeに拡張機能を追加
   - Chrome で `chrome://extensions/` を開く
   - 右上の「デベロッパーモード」をオンにする
   - 「パッケージ化されていない拡張機能を読み込む」をクリック
   - `dist` ディレクトリを選択

## 主な画面

### ポップアップビュー
拡張機能アイコンをクリックすると表示される小さなウィンドウ
- 今日のタスク一覧
- クイックタスク追加
- ミニカレンダー

### フルビュー（オプションページ）
完全なカレンダーインターフェース
- 月/週/日表示
- タスクのドラッグ&ドロップ
- 詳細なタスク編集
- カテゴリフィルター
- 設定画面

## データ管理

### ローカルストレージ
Chrome Storage APIを使用してデータを保存

### エクスポート/インポート
- JSON形式でデータをエクスポート
- バックアップファイルからデータを復元

### データ同期
Chromeアカウントにログインしている場合、複数デバイス間でデータを同期

## トラブルシューティング

### 拡張機能が動作しない場合
1. Chrome拡張機能ページで拡張機能が有効になっているか確認
2. Chrome拡張機能ページでエラーが表示されていないか確認
3. Chromeを再起動

### データが消えた場合
1. 設定画面からデータをエクスポートしておいたバックアップファイルから復元
2. Chromeの同期が有効な場合、別のデバイスからデータが同期される可能性があります
